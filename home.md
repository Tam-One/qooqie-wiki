# Koppeling met Google Tag Manager
Gebruik jij Google Tag Manager voor je website? Dan kan je eenvoudig het qooqie script plaatsen via onze handige tool. Door te klikken op:
![Account Koppelen](/uploads/account-koppelen.png "Account Koppelen")
Koppel je het Google Tag Manager account met je qooqie account. Als je hebt geklikt, krijg je een pop-up en kan je het Google account koppelen.

# Koppeling met Google AdWords
Je qooqie account kun je synchroniseren met je Adwords account. Als je dit doet dan kan je de AdWords accounts die je wilt gebruiken voor dynamische call tracking selecteren.
![Adwords](/uploads/adwords.png "Adwords")
Als je een AdWords account hebt gekozen kan je vervolgens je AdWords campagnes synchroniseren met het qooqie account. Hierdoor hoef je niet alle campagnes opnieuw in te voeren.

# Koppeling met Google Analytics
Als jij je qooqie account koppelt met je Google account, worden je qooqie activiteiten aangemaakt in Analytics als een event. Zo kun je ook in je Analytics terug zien wat jouw qooqie activiteiten zijn!

# Het toevoegen van een campagne
In het kopje “campagne toevoegen” maak je een nieuwe campagne aan. De eerste keuze is of je de campagne actief of inactief maakt. Vervolgens kies je wat voor type nummer je wil, een landelijk of een regionaal nummer. 
Dan kies je de naam van de campagne, zodat jij weet welke campagne je meet. Je kiest een doorschakelnummer zoals al eerder bij de instellingen ingevoerd. 
Het mediatype waarop je je campagne wilt uitzetten is het volgende wat je kiest. 

# Het plaatsen van het script op je website
Het script dat staat weergegeven onder het kopje  “integratie” bij de instellingen plak je in website na de opende body tag. Als je het script aan de broncode van je website hebt toegevoegd kun je onder de tab “domeinen” je domeinnaam toevoegen en valideren. 

# Het vervangen van nummers op je website
Het vervangen van nummers wordt automatisch gedaan door het script zodra deze op je website staat. Het script herkent namelijk de telefoonnummers. Als je de functie inschakelt, dan gaat het script op je website op zoek naar de doorschakelnummers die jij hebt ingegeven bij het qooqie dashboard. Vervolgens vervangt het script deze nummers in een qooqie call tracking nummer. 
Het is ook mogelijk om met een spannertje zelf te kiezen welke telefoonnummers er vervangen dienen te worden op de website. Alle telefoonnummers die je wilt vervangen met een dynamisch qooqie nummer dien je te omhullen met een element naar keuze met het attribuut: class="qooqie-number-placeholder".

# Ik heb meerdere vestigingen, maar wil er maar 1 meten.
Ook dat is mogelijk met qooqie call tracking. Heb jij bijvoorbeeld meerdere vestigingen en maar voor één  vestiging een campagne? Met het attribuut: class=”qooqie-priority-number-placeholder” kun je dit instellen. Op de plekken waar je call tracking wil, plaats je dat stukje script. Op basis van de campagne en de gekoppelde vestiging wordt er een van de dynamische nummers op deze plek getoond.

# Gearchiveerde campagnes
In de gearchiveerde campagnes staan alle campagnes die voorbij zijn. Hier kan je nog een keer de campagnes terug kijken en wat de leads er uit zijn geweest. 

# Campagne overzicht
In het campagne overzicht zie je alle campagnes, actief en inactief. Je kan ook zoeken op een campagne rechts bovenin het scherm. 

# Onsite meten
Door middel van dynamisch gegenereerde nummers kan jij op je website meten waar de inkomende telefoontjes vandaan komen.

# Offsite meten
Door speciaal gegenereerde nummers kun jij ook al je andere campagnes meten, ook degenen vanuit bijvoorbeeld de krant. 
In het dashboard voeg je een nieuwe campagne toe en geef aan dat je een offsite campagne gaat meten. Bijvoorbeeld print, radio of televisie. Vul de rest van de gegevens in en er zal een uniek traceerbaar telefoonnummer gegenereerd worden.

# Wat is een sessie
Een sessie start op het moment dat een bezoeker op jouw website komt en eindigt zodra deze bezoeker jouw website verlaat. 

# Wat is een lead
Zodra iemand op jouw website komt, start er een sessie. Pakt deze bezoeker ook daadwerkelijk de telefoon op en belt dan wordt deze sessie omgezet naar een lead. In de inbox zie je dan onder het kopje “leads” de telefoonnummers die gebeld hebben naar jouw bedrijf. 

Je ziet de status van de oproep, aangenomen of niet. Je ziet of er een opname is gemaakt van het telefoontje. Je ziet het nummer dat heeft gebeld en naar welk nummer er is gebeld. Je ziet vanuit welke campagne de klant gekomen en de duur van het gesprek. Ook wordt het medium weergegeven. Ook zie je welke zoekwoorden er zijn gebruikt en hoe laat iemand gebeld heeft. Je kan er voor kiezen om de opname te downloaden.

# Qooqie advies: het inrichten van jouw account
Ons advies is om je qooqie account te koppelen met je Google Tag Manager account. Dit is de eenvoudigste manier te kunnen beginnen met het meten van je telefonische conversie.
Beoordeling in Google Analytics
In Google Analytics wordt er een event aangemaakt op het moment dat er een qooqie conversie plaatsvind. 

# Keuzemenu configureren
Keuzemenu configureren vind je bij instellingen onder het kopje “telefoonnummers”. Door te klikken op “keuzemenu configureren” krijg je een weergave waar je een aantal gegevens moet invullen. Hier upload je het bestand welke afgespeeld wordt in het keuzemenu. Bij de keuzemenuopties geef je aan naar welk nummer er doorgeschakeld moet worden bij het aanklikken van een bepaald nummer. Ook kies je een nummer waar er naartoe geschakeld moet worden als er geen keuze gemaakt wordt. 

# Call recording
Met call recording kun jij de inkomende telefonie opnemen. De opnames kun je gebruiken voor kwalitatieve doeleinden. Je kunt er zelf voor kiezen of jij telefoontjes wel of niet opneemt. In het instellingenmenu bij “telefoonnummers” kun je aanvinken of je call recording wel of niet aan wilt zetten. 

# Opmaak telefoonnummer
De opmaak van het telefoonnummer kan je zelf kiezen. Dit kan een regionaal of nationaal telefoonnummer zijn. 

# Realtime dashboard
Zodra je ingelogd bent bij qooqie kom je direct bij het Realtime dashboard terecht. Hier zie je de laatste ontwikkelingen in jouw qooqie gegevens.

# Data filteren per periode
Wil je een bepaalde periode terug kijken of een bepaalde dag terug zien? Door te filteren per periode kun je de momenten terug zien die jij terug wilt zien. 

# Nummeruitgave
Nummeruitgave vind plaats vanuit qooqie. Met het door jou gekozen pakket krijg jij een aantal dynamische telefoonnummers die toegewezen zijn aan jouw account. 

# Telefoonnummer type
Het type telefoonnummer kan je zelf kiezen. Je kan kiezen uit een regionaal nummer of een nationaal nummer.

# Regionaal nummer
Met een regionaal telefoonnummer laat je zien vanuit welke regio je komt. Dit zijn bijvoorbeeld 010-nummers. 

# Nationaal nummer
Een nationaal telefoonnummer is een landelijk telefoonnummer waar je niet direct uit kunt afleiden vanuit welke regio je komt. Dit kunnen bijvoorbeeld 085-nummers zijn. 

# Dynamisch nummer
Iedere bezoeker die op jouw website komt, krijgt een eigen uniek telefoonnummer te zien op de website. Hierdoor wordt het inzichtelijk voor jouw bedrijf via welke verkeersbron en zelfs via welk zoekwoord de bezoeker op jouw website is gekomen. 
# Vast nummer
Een vast nummer wordt uitgegeven voor een campagne waarbij er maar één nummer nodig is om te kunnen meten. Dit kan bijvoorbeeld een flyer- of een brochure campagne zijn. 

# Kanalen
Onder de kanalen vind je alle soorten campagnes die je momenteel hebt gemaakt. Je kan per kanaal zien wat er gebeurd. 

# Google
Door de koppeling met je Google account, worden de AdWords campagnes ingeschoten in het qooqie dashboard. 

# Inbox 
In de inbox zie je een overzicht van je sessies en je leads. Ook deze kun je filteren op een periode. 

# KvK nummer wijzigen
Zelf kun je je KvK nummer niet wijzigen. Dit omdat het account opgebouwd is vanuit het KvK nummer. Wil je deze toch wijzigen? Neem dan contact op  met onze klantenservice. 

# Resultaat tot op het zoekwoord (AdWords)
Doordat er aan jouw AdWords advertentie dynamische telefoonnummers gekoppeld worden, is het mogelijk om te achterhalen welke zoekwoorden een bezoeker van jouw website heeft gebruikt. Zo weet je welke zoekwoorden resultaat opleveren en welke minder succesvol zijn. 

# Webbeheerder inlichten
Beheer jij niet zelf je website maar laat je dit door een webbeheerder doen? Dan kan je eenvoudig via deze knop contact opnemen met je webbeheerder.

# De styling van dynamische nummers aanpassen
Je kunt zelf kiezen welke opmaak jouw qooqie telefoonnummers moeten hebben. Ook kun je deze wijzigen. Dit doe je in het qooqie dashboard bij de instellingen onder het kopje Telefoonnummers. Hier kun je kiezen voor een nationale opmaak, internationale opmaak of internationale zonder opmaak kiezen.
